 Titanic-DataCleaning-Task
Overview
This project focuses on cleaning the Titanic dataset using Python (Pandas, NumPy, Scikit-learn) in Google Colab.

Steps Performed
1. Loaded dataset using pandas.
2. Explored dataset shape, columns, and data types.
3. Identified and handled missing values.
4. Dropped high-null columns (like Cabin).
5. Imputed missing numeric and categorical values.
6. Converted categorical features into numerical format using one-hot encoding.
7. Scaled numerical features (Age, Fare) using StandardScaler.
8. Detected and removed outliers using the IQR method.
9. Saved the final cleaned dataset as `Titanic_Cleaned.csv`.

Tools Used
- Python
- Pandas, NumPy, Scikit-learn
- Google Colab
 Files Included
- `Untitled4.ipynb` – Code notebook  
- `Titanic-Dataset.csv` – Original dataset  
- `Titanic_Cleaned.csv` – Cleaned dataset
